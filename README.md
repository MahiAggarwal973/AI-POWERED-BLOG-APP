## AI POWERED BLOG APPLICATION (MERN Stack)
A full-stack AI POWERED BLOG APP built using the MERN stack that allows users to read blogs seamlessly while enabling admins to create, manage, and enhance blog content using AI assistance.
This project focuses on real-world blogging functionality, admin authentication, and scalable backend architecture.

## LIVE WEBSITE :
https://ai-powered-blog-app-client.vercel.app/

## Key Features : 
1. User Features:
* Browse and read published blog posts
* Clean and responsive user interface
* SEO-friendly blog pages
* Fast loading and smooth navigation
2. Admin Features:
* Secure admin authentication
* Create, edit, and delete blog posts
* Manage all blog content from admin panel
* Backend REST API for blog operations
3. AI Features:
* AI-assisted blog content generation or enhancement
* Scalable architecture to integrate more AI tools

## Tech Stack :
1. Frontend:
* React.js
* Context API
* Axios
* React Router
* CSS / Tailwind CSS
2. Backend:
* Node.js
* Express.js
* REST APIs
3. Database:
* MongoDB
* Mongoose
4. Authentication & Security:
* JWT (JSON Web Tokens)
5. Media & Image Optimization:
* ImageKit.io (Image CDN & Optimization)
* Automatic image resizing and compression
* Faster image delivery with CDN support
6. Deployment:
* Vercel (Client & Server)

## Installation & Setup :
1️⃣ Clone the Repository:
git clone https://github.com/MahiAggarwal973/AI-POWERED-BLOG-APP.git
cd AI-POWERED-BLOG-APP
2️⃣ Backend Setup:
cd server
npm install
npm run server
3️⃣ Frontend Setup:
cd client
npm install
npm run dev

## Application Flow
1. Admin logs in securely using JWT authentication
2. Admin creates or updates blog content
3. Blog data is stored in MongoDB
4. Backend APIs fetch blog data
5. Users access and read blogs from the client panel
6. AI assists in content creation or enhancement
